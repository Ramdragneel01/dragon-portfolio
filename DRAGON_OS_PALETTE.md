# Dragon OS Optional Palette Preset

This repository currently uses the default red theme in production.

If you want to use the Dragon OS deep-space palette later, apply the token values below.

## Core Palette

- Deep void: #0D0A1E
- Dark base: #1A1035
- Hero violet: #7C3AED
- Soft lavender: #C4B5FD
- Ice blue: #38BDF8
- Deep purple: #2E1065
- Mid purple: #4C1D95
- Sky blue: #0EA5E9
- Amber pop: #F59E0B
- Off white: #F1F5F9

- :root

## Notes

- This is optional and not active by default.
